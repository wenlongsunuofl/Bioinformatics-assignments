This program is for Project 2.
It is wrote in MatLab R2011a with a 32-bit Windows 7 Operating System.
In order to run this program:
Step1: You need to copy the EM Algorithm folder to your MatLab folder
Step2: Run the 'EM_MotifFinding.m', It will pop up a interface for you, And you can choose sequence file, and  change the parameters here.
Step3: After that, click the 'MotifFinding' button, you will get the result.  Program show the result of last iteration on the interface (on the right)
Step4: The program also store all the results in 'output.txt' in the folder, User can check that too.
